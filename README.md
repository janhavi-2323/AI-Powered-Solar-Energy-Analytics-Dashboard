# AI-Powered Solar Energy Analytics Dashboard

A smart dashboard that predicts solar energy generation using real-time weather data and AI-powered recommendations.

## Features

* Real-time weather search by city
* Solar energy prediction
* AI-powered solar recommendations
* AI Chat Assistant
* Voice input using Speech Recognition API
* Analytics dashboard with charts
* Prediction history tracking
* PDF report generation
* User settings management

## Technologies Used

* React.js
* Tailwind CSS
* OpenWeather API
* OpenRouter AI
* Recharts
* jsPDF

## Installation

```bash
npm install
npm run dev
```

## Deployment

* Frontend: Vercel
* Backend Services: OpenRouter API & OpenWeather API
